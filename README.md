# Skin-Cancer-Malignant-vs-Benign
This project focuses on building a deep learning-based skin cancer classifier using the ISIC (International Skin Imaging Collaboration) dataset. The goal is to assist in early detection of skin cancer by leveraging convolutional neural networks (CNNs) to analyze dermoscopic images.

The model size is above 25MB. It's 25.1MB, so I couldn't upload it. To get the model run the IPython notebook file.
